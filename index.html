Basic Linux Commands:


Shortcut to open terminal in linux press:
ctrl+alt+T

1.ls
Using "ls" command in terminal helps user to know what files are in the present directory(folder).

2.cd
Using "cd" command in terminal helps user to change directory(folder).

3.man (or) --help
Using "man (or) --help" command in terminal helps user to know more about a command and how to use it.

4.cp
Using "cp" command in terminal helps user to copy the files through command line.

5.mv
Using "mv" command in terminal helps user to move the files through command line.

6.cat
Using "cat" command in terminal helps user to dispaly the conents of a file.

7.eog (image viewer)
Using "eog" command in terminal helps user to view images.

8.zip
Using "zip" command in terminal helps user to compress files into a zip archive.

9.unzip
Using "unzip" command in terminal helps user to extract files from a zip archive.

10.reset
Using "reset" command in terminal helps to restore a console to a normal state.

11.exit
Using "exit" command in terminal helps user to exit from terminal.

Important Linux Commands for CTFs:

1.wget
Using "wget" command in terminal helps user to download the
files from the web.

2.ssh
Using "ssh" command in terminal helps user to establish an encrypted secure connection with host machine.

3.nc (netcat)
"netcat" is a utility that reads and writes data across network connections.

4.curl
"curl" is a command that is used to get or send data using URL syntax.

5.xxd
"xxd" is a command that is used to get hex dump of a file.

6.grep
"grep" is a command that is used to search for a paticular characters,which is defined by user.

7.strings
"strings" is a command that used to print the strings of printable characters in given file.

8.fcrackzip
"fcrackzip" is a command that searches each zipfile given for encrypted files and tries to guess the password.
